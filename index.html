Sistema de Asistencias

Este proyecto permite llevar el control de las asistencias de los niños en la Iglesia Palabra de Vida. El cual incluye un lector de huellas que se conecta a una aplicación en Java para tomar la asistencia.

  • Cliente: PDVKids

Sistema CAI

Este sistema funciona para tomar la sesión de los alumnos que asisten al laboratorio de inglés en la universidad y así los teacher puedan acceder al sistema y ver cuantas veces han entrado al laboratorio. El sistema también permite exportar reportes PDF de las sesiones de los alumnos.
El maestro encargado de asignar el proyecto fue: M.S.I. Mario Humberto Rodríguez Chávez.

  • Cliente: Universidad Politécnica de Victoria

Calculadora de Probabilidad y Estadística
